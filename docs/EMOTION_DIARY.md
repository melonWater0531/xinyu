## 二、核心规则

- **一天一个情绪**：选 `😊😢😐😠😨😲🤢😏` 其中一个，代表当天主色调
- **一天一篇日记**：用户自主写入，不限字数
- **日历显示**：每个日期格显示当天情绪小黄脸
- **未写日记的日期**：显示空（但可以点击补写）
- **LLM 回复**：保存日记后自动生成温暖回应

## 三、数据存储

```javascript
// 独立存储 key: xinyu_diary_entries_<用户名>
diaryEntries = [
  {
    _dateStr: "2026-6-11",
    _emoKey: "Happiness",       // 当天唯一情绪
    emotion: "快乐",
    weather: "☀️",              // 用户可选天气
    text: "今天阳光很好...",     // 用户日记正文
    reply: "阳光和好心情...",    // LLM 回复
    editedAt: "14:30"           // 最后编辑时间
  }
]
```

与现有 `diaryEntries` 格式兼容：一个日期一条记录，`_dateStr` 唯一。

## 四、晚间弹窗（9PM，可配置）

```
┌─────────────────────────────────┐
│           🌙 晚间回顾            │
│                                 │
│  今天 6月11日                    │
│  你的心情是 😊 快乐              │
│                                 │
│  今天写了日记 ✍️                 │
│  ┌─────────────────────────────┐│
│  │（LLM 生成的今日回顾）        ││
│  └─────────────────────────────┘│
│                                 │
│  如果今天没写：                  │
│  "今天还没有记录呢，                 │
│   要不要花两分钟写写今天的感受？" │
│       [✍️ 去写]   [💤 明天]     │
└─────────────────────────────────┘
```

触发条件：
- 当前时间 == reminderHour:reminderMinute（默认 21:00）
- 今天还没弹过（localStorage 记录日期）
- 在"我的"页面可设置提醒时间
